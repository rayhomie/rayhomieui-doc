# useDraggable

## 描述

```bash
拖拽
```

## 类型

```ts

useDraggable:(Element) =>  [Element]

传入需要拖拽的元素，返回一个元素
```

## 基础用法

<Playground>
  {() => {
    const [DragElement] = useDraggable(
      <div
        style={{
          width: 100,
          height: 100,
          background: "#0d6efd",
        }}
      />
    );
    return DragElement;
  }}
</Playground>
