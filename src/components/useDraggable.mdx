# useDraggable

## 描述

```bash
拖拽
```

## 类型

```ts
interface Res {
  Ref: React.ReactElement;
  MouseDown: (e: React.MouseEvent) => void;
  MouseMove: (e: React.MouseEvent) => void;
  MouseUp: (e: React.MouseEvent) => void;
}

useDraggable:() =>  Res

Ref、MouseDown、MouseMove、MouseUp需要绑定在拖拽的元素上；
需要提前在拖拽元素上设置style={{ position: "absolute" }}
```

## 基础用法

<Playground>
  {() => {
    const { Ref, MouseDown, MouseMove, MouseUp } = useDraggable();
    return (
      <div
        ref={Ref}
        onMouseDown={MouseDown}
        onMouseUp={MouseUp}
        onMouseMove={MouseMove}
        style={{
          width: 100,
          height: 100,
          background: "#0d6efd",
          position: "absolute",
          zIndex: 99,
        }}
      />
    );
  }}
</Playground>
